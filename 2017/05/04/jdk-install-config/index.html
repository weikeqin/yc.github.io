JDK安装步骤
1.检查是否有JDK，如果有，可以用，直接完成，如果版本低或者没有，如果有open JDK(系统自带的)，可以把open JDK卸载
2.如果有JSE JDK 低版本，可以忽略低版本，也可以删除低版本，安装新版本的JDK，配置/etc/profile文件就行
3.安装JDK，并配置/etc/profile文件就行 Windows系统可以通过环境变量
